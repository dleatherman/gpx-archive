# GPX Viewer

1. Chuck all your `.gpx` files into the `_workouts` folder
2. Run `npm install`
3. Run `npm run start` for development and `npm run build`
4. Deploy your `dist` folder

# Tech Stack

- [Eleventy](https://11ty.dev/)
- [SVGO](https://github.com/svg/svgo)
- [D3](https://github.com/d3/d3)
- [Sparkline-SVG](https://github.com/quisido/sparkline-svg)
- [topojson](https://github.com/topojson/topojson-simplify)
- [@mapbox/togeojson](https://github.com/mapbox/togeojson)
